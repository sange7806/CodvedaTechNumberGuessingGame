# CodvedaTech Number Guessing Game

This project is a simple number guessing game where the user attempts to guess a randomly generated number within a limited number of attempts.

## 🎥 Demo Video
Watch the demo here:
https://drive.google.com/file/d/1xRbKdSSTAsz7f6QYVGT72RO9rT-xducB/view?usp=sharing

## 🚀 Features
- Random number generation
- User input handling
- Feedback (Too high / Too low)
- Limited attempts
- Win/Lose outcome

## 📚 Concepts Covered
- Loops
- Conditional statements
- Random number generation
- User interaction

## 🛠️ Technologies Used
- Java
